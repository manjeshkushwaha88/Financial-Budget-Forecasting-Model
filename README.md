# 📊 Financial Budget Forecasting Model

## 🔹 Project Overview

This project presents a **scenario-based financial forecasting model** built in Microsoft Excel. It evaluates how changes in key financial drivers impact revenue, costs, EBITDA, and net profit under different business scenarios.

The model is designed to support **financial planning, budgeting, and strategic decision-making**.

---

## 🎯 Objectives

* Develop a dynamic financial forecasting model
* Perform scenario analysis (Base, Best, Worst cases)
* Forecast revenue, costs, and profitability
* Support data-driven financial decisions
* Demonstrate financial modeling skills
