## 📊 Financial Forecasting Model

This project presents a dynamic financial forecasting model built in Excel. It enables users to analyze key financial statements and evaluate different business scenarios using adjustable input assumptions.

---

## 🚀 Model Overview

The model includes:

* ✔️ Input assumptions for scenario planning
* ✔️ Automated income statement projections
* ✔️ Financial forecast table
* ✔️ Clean and structured Excel model design

> ⚠️ **Note:** GitHub does not preview Excel files. Please download the Excel file from this repository to interact with the full model.
